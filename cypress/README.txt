INSTRUCCIONES DE EJECUCIÓN

1. Clonar repositorio
git clone <url_repo>

2. Instalar dependencias
npm install

3. Ejecutar pruebas
npx cypress open

4. Ejecutar test
purchase.cy.js

La prueba automatiza un flujo E2E en Demoblaze que incluye:
- Agregar productos al carrito
- Validar carrito
- Completar formulario
- Finalizar compra
